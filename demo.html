Brro FX v1.0.0

Motion effects
via CSS classes.

A lightweight WordPress plugin delivering Elementor-style scroll effects with zero dependencies. Add a class, get an effect. Stack as many as you like.

Fade (Opacity Mapping)

Opacity maps to the element's scroll progress. The element fades in as it enters the viewport. Add brro-fx--direction-down to invert (fade out). Use brro-fx--start-* / brro-fx--end-* to lock the fade to a specific portion of the journey.


brro-fx--fade
fades in as scrolling up

brro-fx--fade
brro-fx--direction-down
fades out

brro-fx--start-20
brro-fx--end-60
restricted range (20%→60%)

Blur (Gaussian Blur Mapping)

Rotate

The element rotates as it passes through the viewport. At 50% progress (element centre aligned with viewport centre) rotation is 0°. Speed controls the total degree range (speed 5 → ±50°).


brro-fx--rotate
clockwise (default)

brro-fx--speed-3
counter-clockwise, slow

Stacking

Combined Effects

Multiple effects can coexist on the same element. Each writes to its own CSS custom property; a single shared transform / filter / opacity rule combines them — no conflicts.

brro-fx--vertical-scroll
brro-fx--fade
brro-fx--speed-4 parallax + fade-in
brro-fx--vertical-scroll
brro-fx--rotate
brro-fx--blur
brro-fx--speed-5 parallax + rotate + blur
brro-fx--horizontal-scroll
brro-fx--fade
brro-fx--direction-right
brro-fx--speed-6 horizontal + fade-out

A looping two-pulse scale animation. Pure CSS — no scroll tracking needed. Combine speed and intensity variants freely.

brro-fx--heartbeat default (2s, scale 1.1)
brro-fx--heartbeat
brro-fx--speed-400ms fast pulse
brro-fx--heartbeat
brro-fx--scale-0-8 shrink pulse
brro-fx--heartbeat
brro-fx--speed-400ms
brro-fx--scale-0-8 fast shrink
❤️ brro-fx--heartbeat works on any element

Page Tracking

brro-fx--page

Add brro-fx--page to any scroll-effect element and progress maps to the full page scroll (0% = page top, 100% = page bottom) instead of the element's own viewport position. The progress bar below uses this.

Speed & Range

ClassEffectExamples
brro-fx--speed-{1–10}Controls effect magnitude for scroll effects. Default: 5.vertical: ±N×20 px, blur: N×2 px max, rotate: ±N×10°
brro-fx--start-{0–100}Progress % at which animation begins. Default: 0.brro-fx--start-20 → effect starts at 20% progress
brro-fx--end-{0–100}Progress % at which animation completes. Default: 100.brro-fx--end-70 → fully animated by 70% progress

Smooth Scroll

AttributeEffect
<html brro-fx--smoothscroll="on"> Enables lerp-based smooth scroll (easing factor 0.1) for wheel events. Automatically skipped when prefers-reduced-motion is set. Keyboard, scrollbar, and anchor-link scrolls pass through as native.

Stacking Example

Any number of scroll effects can be combined on one element:

<div class="brro-fx--vertical-scroll
             brro-fx--fade
             brro-fx--blur
             brro-fx--speed-4
             brro-fx--start-10
             brro-fx--end-80">
  All three effects, restricted to the 10%–80% progress window.
</div>
