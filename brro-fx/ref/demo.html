Brro FX v1.1.0

Motion effects
via CSS classes.

A lightweight WordPress plugin delivering Elementor-style scroll effects with zero dependencies. Add a class, get an effect. Stack as many as you like.

Per-Effect Parameters

Different parameters per effect

Each effect can take its own direction, start, end and speed via namespaced classes: brro-fx--{effect}-{param}-*. Scoped classes win over the legacy globals (brro-fx--start-*, brro-fx--speed-*, etc.), which still work as fallbacks. Loop Animation

Heartbeat

Looping scale pulse: scale(1) at the start/end of each cycle and var(--brro-fx--scale) at the midpoint (default 0.95 — gentle shrink). Pure CSS — no scroll tracking. Add brro-fx--duration-* (numeric, e.g. brro-fx--duration-400) for cycle length and brro-fx--scale-* for the midpoint scale.

Entrance Animation

Fade In

Triggered once when the element enters the viewport (10% threshold). The brro-fx--is-visible class is added by the engine; the CSS transition does the rest. This triggers once per page load (it does not replay on scroll back up). Two variants: brro-fx--fade-in (opacity) and brro-fx--fade-in-up (opacity + 24px slide-up). Optional: brro-fx--delay-* and brro-fx--duration-* (defaults are 0.65 s per property).

Per-Effect Modifiers (v1.1.0+)

Scoped variants of the global direction, start, end and speed classes. Each effect uses its own scoped class first; if none is present, the corresponding global class is used, otherwise the default applies. Scoped > global > default. Fade is renamed for clarity: brro-fx--fade-direction-in (default) / -out (replaces the global direction-down meaning for fade).

EffectDirectionStart / EndSpeed
brro-fx--vertical-scroll brro-fx--vertical-direction-up (default) / -down brro-fx--vertical-start-{0–100} / -end-{0–100} brro-fx--vertical-speed-{1–10}
brro-fx--horizontal-scroll brro-fx--horizontal-direction-left (default) / -right brro-fx--horizontal-start-{0–100} / -end-{0–100} brro-fx--horizontal-speed-{1–10}
brro-fx--fade brro-fx--fade-direction-in (default) / -out brro-fx--fade-start-{0–100} / -end-{0–100} —
brro-fx--blur — brro-fx--blur-start-{0–100} / -end-{0–100} brro-fx--blur-speed-{1–10}
brro-fx--rotate brro-fx--rotate-direction-right (default, CW) / -left (CCW) brro-fx--rotate-start-{0–100} / -end-{0–100} brro-fx--rotate-speed-{1–10}

Note: --brro-progress still reflects the global brro-fx--start-*/-end-* window (unchanged for backward compatibility). Each effect internally uses its own clamped progress from the same raw scroll position.

Smooth Scroll

AttributeEffect
<html brro-fx--smoothscroll="on"> Enables smooth scrolling via Lenis 1.1.19 (bundled and auto-loaded as a dependency). Defaults: duration: 1.2 s, exponential easing, wheelMultiplier: 1, smoothTouch: false (touch devices use native scroll). Automatically skipped when prefers-reduced-motion is set.

Duration utilities (brro-fx--duration-*)

Generic animation-duration and transition-duration. Class names use a numeric suffix only (e.g. brro-fx--duration-800 = 800 ms) — no ms in the selector — to keep them distinct from scroll brro-fx--speed-{1–10} (JS only reads digit-only speed classes). In this plugin’s CSS, these rules are declared after brro-fx--heartbeat and the fade-in shorthands so the longhands win the cascade (same class specificity — order matters; no !important). Default heartbeat cycle is 4000 ms; add any row below to shorten or lengthen it (or time fade-in / custom CSS).

Stacking Example

Any number of scroll effects can be combined on one element. With global classes, a single window drives all effects:

<div class="brro-fx--vertical-scroll
             brro-fx--fade
             brro-fx--blur
             brro-fx--speed-4
             brro-fx--start-10
             brro-fx--end-80">
  All three effects share the same 10%–80% progress window.
</div>

With per-effect classes, each effect has its own direction / window / speed:

<div class="brro-fx--vertical-scroll
             brro-fx--vertical-direction-up
             brro-fx--vertical-start-20
             brro-fx--vertical-end-60
             brro-fx--fade
             brro-fx--fade-direction-out">
  Vertical drifts up only from 20%–60% of the journey;
  fade runs its default 0%–100% window and fades out.
</div>
